Comunicación del Nuevo Planeta: "Melmac"

Desarrollo de sistemas de comunicación avanzados que permitan la interacción fluida entre este nuevo territorio y la Tierra.

Al tener que salir del planeta para habitar uno diferente, el sistema de comunicación entre nuestro planeta y el nuevo planeta se vuelve vital para poder tener registros constantes de como va el proceso de descontaminación ya que este planeta sería un paso intermedio mientras la tierra poco a poco en futuro se vuelva habitable, para ello creamos un dron con la capacidad de registrar y recolectar muestras de nuestro planeta con la finalidad de que en un futuro este se vuelva nuevamente habitable.
